Website Redefine Multi Images Position

You must have noticed some e-commerce websites having multi images. Website Redefine Multi-Images Position is used to change the position of multi images tab as left, right, top, or bottom. Generally, multi images are shown in the bottom but with the help of this module, multi image tab can be shown in left, right, top, or bottom according to the backend configuration.
