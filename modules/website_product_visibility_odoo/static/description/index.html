Website Product Visibility Per Customer Or Guest

With the help of this module you can visible / invisible any product, product category per customer or for guest on webshop. User have Administrator Setting access can set which product will visible for any user or guest user.

Work With Both Community And Enterprise Version.

Key features:
  • Easy To Use.
  • User have Administrator --> Setting access can set which product will visible for logged in user.
  • User have Administrator --> Setting access can set which product will visible for guest user.
  • User have Administrator --> Setting access can set which product category and all product from that category will visible for logged in user.
  • User have Administrator --> Setting access can set which product category and all product from that category will visible for guest user.
  • Product can be easily select from Sale Order history.
  • Setting is per customer so a product can be visible to one user but not visible to another user.
After Installation of app all product will hide from Guest User/Any User.
Guest User Visibility Setting
Goto --> Website --> Products --> Public Products.
  • You can manually add Product / Product Category one by one.
  • You can click on ADD ALL PRODUCT, ADD ALL CATEGORIES button to add all products or all products categories at a time.
  • All products from selected product category will visible to webshop for guest user.
Product visibility setting for particular user
Goto --> Contacts --> Select Contact --> Website Visible Products page.
  • You can manually add Product / Product Category one by one.
  • You can click on ADD ALL PRODUCT, ADD ALL CATEGORIES button to add all products or all products categories at a time.
  • You can click on ADD PRODUCT FROM SALE HISTORYbutton to select products from Sale Order history.
  • All products from selected product category will visible to webshop for current User user.
User does not have Administrator --> Setting access, can not set which product / product category to display.
