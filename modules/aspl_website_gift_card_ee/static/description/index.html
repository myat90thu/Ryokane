Odoo 12 Website Gift Card (Enterprise)

This module allows user to purchase gift card,use gift card and also recharge gift card.


Note: This module is tested and working in Odoo enterprise edition And we are supporting Ubuntu OS and not Windows OS.

Features

  • Allow user to buy gift card.
  • Allow user to redeem gift card.
  • Allow user to recharge gift card.
  • Allow user to see gift card details, usage details and recharge details.

Available for Community

How to use

  • User can buy or see details of gift card.
  • User can buy gift card from this page.
  • User can check gift card details from this page.
  • User can view card balance, card history. Also user can rechange card and change pin of card from this page.
  • User can view card recharge history from this tab.
  • User can redeem gift card from below steps.
  • After redeem gift card user see this page.
  • After buy gift card user receive mail regarding gift card number and link for set pin.
  • User can set pin for gift card from this page
  • After recharge gift card user receive mail for successfully done recharge.
